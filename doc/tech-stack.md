# Tech Stack (목록)

- **Language/Framework**: Java 17, Spring Boot 3.x, Spring Data JPA, Validation
- **DB/Infra**: PostgreSQL, PostGIS(거리 계산), Flyway(마이그레이션), Redis(블랙리스트/Streams)
- **Auth**: JJWT 기반 JWE Access/Refresh, Security Filter 인증
- **Storage**: S3-compatible (AWS S3 또는 MinIO Presign/Confirm)
- **Messaging/Push**: Firebase Cloud Messaging (FCM)
- **Build/Tools**: Gradle, JUnit/Mockito, Mermaid, Markdown 문서화
