# 리팩토링 후보 (현 시점)

- **알림 워커 고도화**: 재시도/백오프 정책 일원화, notification_targets 상태 업데이트 정교화.
- **알림 이력 조회 API**: 사용자별 알림 조회/검색, 오류율 모니터링 지표 노출.
- **매출 엑셀 Export**: 비동기 생성 + Presigned URL 제공 (대량 데이터 대응).
- **만료 처리 배치**: `no_show_posts` 자동 만료 스케줄러 추가.
- **DTO/Validation 정리**: 공통 응답/요청 규격 통일 및 Validation 메시지 일원화.
- **권한/보안 강화**: 관리자/사장/소비자 역할별 세분화 및 리소스 레벨 접근 제어.
